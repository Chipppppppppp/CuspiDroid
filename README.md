# CuspiDroid

CuspiDroid is an Android 5ch viewer with a browser-like tab interface.

## Features

- Open multiple tabs like a browser
- Enter a URL directly
- Search threads through `find.5ch.net`
- Render 5ch thread URLs in a native reader view
- Fall back to WebView for search pages and regular web pages

## Build

Open this repository in Android Studio, or run:

```powershell
.\gradlew.bat assembleDebug
```
